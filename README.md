# 🛒 Superstore Sales Analysis

## 📌 Project Overview
Analyzed 9,994 sales transactions from a US Superstore to uncover 
business insights on revenue, profit, regional performance, and 
the impact of discounts.

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- Google Colab
- Dataset: Superstore Sales Dataset (Kaggle)

## 🔍 Key Business Questions Answered
1. Which category generates the most revenue?
2. Which category is most profitable?
3. Which region performs best?
4. What does the sales trend look like over time?
5. Does offering discounts hurt profit?

## 📊 Key Findings
- **Technology** is the highest revenue and most profitable category
- **Furniture** generates strong revenue but very low profit
- The **West region** is the most profitable
- Sales peak every **November/December**
- Discounts above **30% consistently cause losses**
